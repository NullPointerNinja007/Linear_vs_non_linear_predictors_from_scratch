# Heart Disease Prediction with Machine Learning

This project compares a linear classifier (logistic regression) and a shallow neural network for predicting heart disease using clinical data.

The models are implemented **from scratch using NumPy** to explore the differences between linear and nonlinear decision boundaries.

## Dataset

UCI Heart Disease Dataset.

Features include:

- age
- cholesterol
- resting blood pressure
- maximum heart rate
- chest pain type

Target:

0 = no heart disease  
1 = heart disease

## Models

1. Logistic Regression
2. 1-Hidden-Layer Neural Network

## Goals

- Evaluate how well linear models perform on medical risk prediction
- Investigate whether a shallow neural network improves performance
